# Changelog

All notable changes to Stage Mirror are documented in this file.

## 0.1.0 - 2026-07-22

### Added

- **Restore Working Tree from HEAD** in the built-in Git SCM resource context menu.
- Multi-selection, duplicate removal, multi-root, nested-repository, and multi-repository support.
- Shell-free Git execution with repository-relative pathspecs.
- Confirmation and success-notification settings.
- Partial-failure reporting and best-effort Source Control refresh.
- Strict TypeScript and ESLint configuration with all runtime logic in one source file.
